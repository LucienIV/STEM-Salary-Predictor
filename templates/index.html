What is this webpage?

This is a webpage created to display trends shown in salary data for working professionals in STEM fields of employment. Using data scraped from the salary comparison website levels.fyi further analysis was made to the data. The data was then cleaned using Python and pandas into a form usable for visualization and machine learning. Trends and distributions were visualized using Tableau and are viewable under the Graphs section of the webpage. Machine learning methods were applied to the data and a model was created to predict salary range based on supplied information. This can be found on the Predictor section of the webpage.

  • Software Engineering Manager was the highest paying profession among the STEM professions listed
  • California is the state with the highest average base salary
